## 주요 패턴 분석

### 1. 설정 관리 패턴
- Vue 설정 컴포넌트들은 주로 `routes/setting.py`와 연결
- 저장: `POST /savefile/{channel}` 패턴 사용
- 조회: `GET /getSettingData/{channel}` 패턴 사용

### 2. 인증 관리 패턴
- 사용자 관리는 `routes/auth.py`와 연결
- CRUD 패턴: join(생성), getUserList(조회), saveUser(수정), removeUser(삭제)

### 3. 자산 관리 패턴
- 자산 관련 기능은 `routes/setting.py`와 연결
- 등록/해제: registerAsset/unregisterAsset
- CRUD: createAsset, modifyAsset, deleteAsset

### 4. 트렌드 데이터 패턴
- 트렌드 관련 기능은 `routes/api.py`와 연결
- 파라미터 조회 후 데이터 요청하는 2단계 패턴

이 매핑 테이블을 통해 Vue 컴포넌트와 FastAPI 라우터 간의 정확한 연결 관계를 파악할 수 있습니다.
