<< Ubuntu Flask Background 가상환경 실행>>

1.tmux 설치하여 사용하면, 가상환경에서 실행한 flask 가 putty 접속이 끊어져도 계속 실행됨
sudo apt install tmux

2. tmux 세션 생성
tmux new -s flask

3. flask 실행
cd ~/WebProject  # Flask 프로젝트 폴더로 이동
source venv/bin/activate  # 가상환경 활성화
python3 main.py  # Flask 실행

4. putty 재접속 후, flask 세션 복귀
tmux list-sessions
tmux attach -t flask

5. (tmux) tmux detach 로 강제탈출 ( 세션은 계속 실행 중)

6. (tmux) exit 세션 나가면서 flask 세션 종료

7. tmux 세션 강제종료
tmux kill-session -t flask
