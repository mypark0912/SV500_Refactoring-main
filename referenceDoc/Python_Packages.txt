<putty 재접속 후, flask 세션 복귀>
tmux list-sessions
tmux attach -t flask

(tmux) tmux detach 로 강제탈출 ( 세션은 계속 실행 중)

(tmux) exit 세션 나가면서 flask 세션 종료

<tmux 세션 강제종료>
tmux kill-session -t flask
