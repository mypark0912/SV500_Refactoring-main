## SmartSystem
- 정격 주파수를 50Hz 및 60Hz로 제한
- SV500 tar 파일로부터 복원(Restore) 기능 구현
- Monitor/Waveform에 파일 불러오기(Load File) 기능 구현
- GUI(Monitor App)에 NarrowBand 기능 구현
- FFT 피크를 3개 영역으로 탐색하는 기능 추가
- Slot Passing Harmonics(동적 편심) 계산 오류 수정
- VSignal 및 ISignal에 시그니처 추가
- Switching peak를 Peak에서 PeakPair로 변환하도록 개선
- 소음 및 진동 고조파를 25차 고조파까지로 제한
- 구동측 속도가 다를 경우 문제가 발생하는 SLOT 피크 선택 기능 비활성화 처리
- Resistance Reactance 제거. Rectifier FM: 불평형 제거. Cable Fault: stator unbi 대신 unbiNema만 적용
- Parse 함수의 채널 순서 유효성 검증 개선
- Narrowband Reporting을 위한 Signal 수신 메서드 추가
- DCLink 측대역(sidebands)이 대칭이어야 하는 버그 수정
- Dynamic Ecc.에서 피크 쌍 검사 오류 수정 (라인 주파수 주변이 아닌 rb*rf +/- rf 주변에 위치하므로)
- Processed 폴더에 최근 이벤트 파일 50개, 웨이브 파일 20개(하루 4개) 유지하도록 개선. 약 5일분 보관.
